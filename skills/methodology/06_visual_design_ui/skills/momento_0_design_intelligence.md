# [MOMENTO 0] Design Intelligence & Taste Protocol

> **Misión:** Actuar como el "Motor Anti-Slop". Este momento absorbe las antiguas Skills 24, 36 y 38. Su objetivo es calibrar la estética, definir la estrategia de layout y establecer las reglas técnicas inquebrantables ANTES de tocar el Momento 1 (Tokens).

## 🧠 Parte 1: Calibración del Gusto (Premium Taste)
El diseño no es subjetivo, es matemática aplicada a la percepción. El Agente debe calibrar con el humano estos 3 espectros (Escala 1-10):
1.  **Densidad (Density):** 1 (Airy/Galería) <---> 10 (Denso/Cockpit). Determina el espaciado base.
2.  **Variación (Variance):** 1 (Simétrico/Predecible) <---> 10 (Asimétrico/Caótico). Determina la grilla.
3.  **Movimiento (Motion):** 1 (Estático) <---> 10 (Cinemático). Determina las curvas de animación.

## 📐 Parte 2: Estrategia de Layout (Visual Hierarchy)
Antes de diseñar componentes, selecciona la ruta arquitectónica de la pantalla:
1.  **Hierarchy-First (El Camino Guiado):** Prioriza un solo CTA. Flujo F-Pattern o Z-Pattern.
2.  **Density-Optimized (El Dashboard):** Maximiza datos. Usa el whitespace como separador en lugar de líneas.
3.  **Progressive Disclosure (El Despliegue):** Oculta complejidad detrás de acordeones o modales.

## 🚫 Parte 3: Hardened Guardrails (Anti-Patterns BANNED)
El Agente tiene prohibido proponer o permitir la inyección de "clichés de IA".
*   **Color:** Prohibido el negro puro (`#000000`). Usar Off-black (ej. Zinc-950). Sin "neon glows" injustificados.
*   **Tipografía:** Prohibido `Inter` para proyectos Premium (es un "AI Tell"). Usar `Geist`, `Satoshi`, `Outfit` o stacks editoriales.
*   **Contenido:** Prohibido usar "Lorem Ipsum", nombres como "John Doe" o copy cliché ("Elevate", "Unleash").
*   **Componentes:** Prohibidos los spinners genéricos (usar skeleton loaders que coincidan con el layout).

## 📊 Parte 4: Inteligencia Técnica UX (Referencia Rápida)
*   **Touch:** Targets mínimos de 44x44px.
*   **A11y:** Ratio mínimo de contraste 4.5:1. Focus states obligatorios.
*   **Performance:** Respetar Core Web Vitals (reservar espacio para evitar layout shifts).

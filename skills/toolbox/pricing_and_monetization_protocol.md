# Skill 15: Protocolo de Precios y Monetización (Pricing Strategy)

Guía técnica para el diseño de modelos de ingresos, empaquetado de funcionalidades y estrategias de captura de valor.

## 🎯 Objetivo Estratégico
Diseñar estructuras de precios que alineen el valor entregado con la disposición a pagar del usuario (Willingness to Pay), optimizando la conversión y el ARPU.

## 🧠 Conceptos Clave
- **Métricas de Valor:** ¿Qué cobramos? (Por usuario, por uso, por transacción, tarifa plana). Debe escalar con el éxito del cliente.
- **Good-Better-Best:** Marco de trabajo para tiers.
  - **Good:** Core funcional para entrada.
  - **Better:** La opción recomendada (Ancla).
  - **Best:** Premium con funcionalidades avanzadas y soporte dedicado.
- **Psicología de Precios:** Uso de anclajes, efecto señuelo y precios redondos para productos premium.

## 🛠️ Protocolo de Ejecución
1. **Identificación de Persona:** ¿Para quién es cada tier? (SMB, Mid-market, Enterprise).
2. **Investigación Van Westendorp:** Determinación del rango de precios aceptable (Demasiado caro vs. Demasiado barato).
3. **Diferenciación de Tiers:** Límites por uso, gating de funcionalidades y niveles de servicio (SLA).

## 📋 Checklist de Monetización
- [ ] Definida la métrica de valor que escala con el cliente.
- [ ] Diferenciación clara entre tiers (que no compitan entre sí).
- [ ] Estrategia de descuento anual definida (17-20%).
- [ ] Plan de "Grandfathering" para clientes existentes en caso de suba de precios.
